- ingresar a discord dev para poder crear una aplicacion y luego un robot

- en "robot" hay un token el cual deberas copiar, este es como la contraseña de tu bot.. 
    tienes que poner ese token en tu código ' bot.run("<aqui va el token>") '

- en "OAuth2" hay una caja la cual contiene una opcion llamada "robot", seleccionamos y debajo apareceran los permisos que tendra el bot

- entre las cajas habrá un código de url el cual copiaremos y abriremos en una pestaña nueva, esto es para finalmente asignar el robot a uno de tus servidores
